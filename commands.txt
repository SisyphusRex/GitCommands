To Set up Username and email after downloading git bash:
    git config --global user.name "<name>"
    git config --global user.email <email>

Change Text Editor
    git config --global core.editor <notepad>

Copy:
    ctrl + insert
Paste:
    shift + insert

To create a git repository:
    1. Create repository on Github
    2. Copy repo url
    3. Open Gitbash
    4. Move to appropriate directory
    5. git clone <url>
    6. git pull

To push to github:
    1. git add -A
    2. git commit
        add comment in notepad
        OR git commit -m "<commit message>"
    3. git push

To Push New Branch:
    git push -u origin <branch name>

Show commit history:
    git log

Undo changes since last commit:
    git checkout .

Discard local changes to all files:
    git reset --hard

Create Branch:
    1. git branch <name>
    2. git checkout <name>
    OR
    1. git checkout -b <name>

Merge branch to main:
    1. Move to main
        git checkout main
    2. git merge <branch name>

Show Branches:
    1. git branch -a

Delete branch:
    1. git branch -d <name>

Paste
    shift + insert

Add .gitignore
    touch .gitignore
    code .gitignore

reset git add before commit
    git reset

show current staged files
    git status

Virtual Environment:
    1. get in project directory
    2. Create
        python -m venv .venv
    3. Activate
        . .venv/scripts/Activate
    4. install packages/do work
        python -m pip install <package>
    5. save requirements
        python -m pip freeze > requirements.txt
    6. deactivate
        deactivate

    install venv from requirements:
        python -m pip install -r requirements.txt
